# Correção do tema escuro — Subcomponentes

Esta atualização corrige a legibilidade do tema escuro.

## O que muda

- Textos dos cards ficam claros no tema escuro.
- Títulos das tabelas ficam claros.
- Valores dos indicadores ficam claros.
- Labels dos filtros e formulários ficam claros.
- Menu lateral selecionado passa a ter contraste correto.
- Inputs continuam com fundo claro para facilitar digitação.

## Como aplicar

Substitua no GitHub apenas este arquivo:

```txt

```
