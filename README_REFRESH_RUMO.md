# Refresh visual Rumo — Subcomponentes

Esta entrega aplica ao sistema de Subcomponentes o mesmo padrão visual usado no refresh do sistema de Dormentes, para que os dois pareçam produtos da mesma família.

## O que foi alterado

- Logos oficiais Rumo em `assets/brand`.
- Favicon Rumo em `index.html` e `login.html`.
- Login redesenhado com fundo azul institucional, grafismos de movimento e card com faixa multicolorida.
- Sidebar no mesmo padrão do outro sistema: azul profundo, linhas/grafismos, logo negativo e rodapé institucional.
- Topo, filtros, cards, KPIs, tabelas, badges, botões e hover states alinhados à identidade Rumo.
- Tema escuro revisado para preservar a paleta Rumo e a legibilidade.
- Cores dos gráficos internos ajustadas para a paleta institucional.

## Paleta usada

- Azul Rumo: `#003865`
- Azul escuro: `#002D51`
- Azul claro: `#32A6E6`
- Verde: `#1E9F7F`
- Verde claro: `#7FE06C`
- Amarelo: `#FBD300`

A lógica do sistema, Supabase e regras de negócio foram preservadas. As alterações foram focadas em aparência, identidade visual e consistência entre sistemas.
