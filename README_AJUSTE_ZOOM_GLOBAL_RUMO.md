# Ajuste visual global — zoom out Rumo

Esta versão reduz a escala visual do sistema inteiro para evitar a sensação de zoom alto.

## Alterações principais

- Redução global de fontes, paddings, gaps e alturas internas.
- Dashboard reorganizado com KPIs mais compactos.
- Card de Saldo em estoque ajustado para não cortar números grandes.
- Filtros, cards, gráficos e tabelas com densidade visual menor.
- Mantido o padrão Rumo: paleta, grafismos, bordas, cards e identidade visual.
- Cache atualizado no `index.html` para forçar o navegador a buscar o CSS novo.

Nenhuma regra de negócio, Supabase ou estrutura de dados foi alterada.
