# Atualização — Remover card superior das páginas

Esta atualização remove o card superior com o texto "Controle de qualidade de subcomponentes" de todas as telas do site.

## Arquivo para substituir

- `js/app.js`

## O que muda

- O card/hero superior deixa de ser renderizado em todas as páginas.
- Nenhum dado do Supabase é alterado.
- Nenhuma tabela, usuário, auditoria ou permissão é modificada.

## Como aplicar

1. No GitHub, abra a pasta `js`.
2. Substitua o arquivo `app.js` por este arquivo atualizado.
3. Faça commit.
4. Aguarde o GitHub Pages atualizar.
5. Abra o site e pressione `Ctrl + F5`.
